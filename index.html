Bash-includes

Scripts for a shellscript includes library to ease some aspects of bash shellscripting

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Bash Includes Library, a number of bash shellscript include files to ease script writing by mimicking nawk or C function calls, not to invent frequent used tools over and over. The library also is to promote more structured script programming.
